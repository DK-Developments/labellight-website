Printing from Product Pages

The most common way to print labels is directly from individual product pages in Lightspeed Retail.

Single Product Labels

  1. Navigate to Products in Lightspeed Retail
  2. Click on any product to open its detail page
  3. Look for the Print Label button added by Label Light (usually in the action bar)
  4. Click the button to open the print modal

Products with Variants

If a product has variants (e.g., different sizes or colors), Label Light will show all variants in the print modal. You can:

  • Print labels for all variants at once
  • Adjust the quantity for each variant individually
  • Set quantity to 0 to skip a variant

Printing from Consignments

Label Light makes it easy to print labels for all products in a consignment (purchase order), perfect for labeling incoming inventory.

Opening a Consignment

  1. Navigate to Inventory ‚Üí Consignments in Lightspeed Retail
  2. Click on a consignment to open it
  3. Look for the Print Labels button (Label Light adds this next to the existing print button)

Print Modes for Consignments

For partially received consignments (where only some items have arrived), Label Light offers special print modes:

  • Print All - Print labels for all products in the consignment
  • Print Received Only - Print labels only for products that have been marked as received
  • Print Unreceived Only - Print labels for products still pending receipt

Bulk Printing from Product Lists

Print labels for multiple products at once from the main products list view.

Selecting Products

  1. Navigate to Products in Lightspeed Retail
  2. Use the checkboxes to select the products you want to print labels for
  3. Look for the Print Labels button that appears in the toolbar when products are selected

Bulk Print Options

The bulk print modal shows all selected products. You can:

  • Review all products before printing
  • Adjust quantities for individual products
  • Remove products from the print batch
  • Print all at once

Large Print Jobs

For very large print batches (100+ labels), the printing process may take a few minutes. Label Light shows progress during printing and allows you to cancel if needed.

Understanding the Print Modal

The print modal is where you configure and execute your label printing. Here's a breakdown of its components:

üñºÔ∏è

Screenshot: Annotated print modal showing all components

Image filename: print-modal-annotated.png

Modal Components

  1. Header - Shows the page title and close button
  2. Printer Selection - Dropdown to choose your printer (if multiple are available)
  3. Template Selection - Choose which label template to use
  4. Label Preview - Live preview of how the label will look
  5. Product Table - List of products/variants to print with quantity controls
  6. Footer - Printer status and Print button

Label Preview

The label preview shows exactly how your label will look when printed.

Preview Features

  • Live Updates - The preview updates as you hover over different products/variants
  • Actual Data - Shows real product data, not placeholder text
  • Template Switching - Preview updates when you change the label template
üñºÔ∏è

Screenshot: Label preview panel showing a rendered label

Image filename: label-preview-panel.png

Template Selection

If you have multiple label templates uploaded, you can switch between them:

  1. Click the Template dropdown above the preview
  2. Select a different template
  3. The preview will update to show the new template

Quantity Controls

Control how many labels to print for each product or variant.

Adjusting Quantities

  • Use the + and - buttons to adjust quantity
  • Click directly on the number to type a specific quantity
  • Set to 0 to skip that item
üñºÔ∏è

Screenshot: Quantity controls in the product table

Image filename: quantity-controls.png

Consignment Quantities

When printing from consignments, the default quantity is set to match the ordered or received quantity. You can still adjust these before printing.

Total Labels Count

The footer shows the total number of labels that will be printed based on your quantity selections. This helps you know how many labels to have ready.

Exporting Product Data

Label Light can export product data to CSV format for use in other applications or for record-keeping.

Export Options

When viewing the print modal, look for the Export button. You can export:

  • Table Data Only - Export just the data shown in the current table
  • All Product Data - Export complete product data including fields not shown in the table
üñºÔ∏è

Screenshot: Export options dialog

Image filename: export-options.png

Export File Format

Data is exported as a CSV (Comma Separated Values) file that can be opened in:

  • Microsoft Excel
  • Google Sheets
  • Apple Numbers
  • Any text editor
üí°

Use Case

Exporting is useful for keeping records of what labels were printed, or for importing product data into other systems like inventory management tools.
