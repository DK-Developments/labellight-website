Printer Setup

Configure your default DYMO printer and detection settings.

Selecting Your Default Printer

If you have multiple DYMO printers connected, you can choose which one to use by default:

  1. In the General Settings section, locate the Printer dropdown
  2. Click the dropdown to see all detected DYMO printers
  3. Select your preferred printer
  4. Click Save Settings

Some versions of DYMO Connect have a bug where printer detection always returns false, even when a printer is connected. If you're experiencing this issue:

  1. Enable the Bypass Printer Detection checkbox
  2. Click Save Settings

When to Use This

Only enable this option if Label Light reports "No printers found" but your printer is working correctly in DYMO Connect software. This bypasses the connection check and assumes your printer is available.

API Token Configuration

For advanced features like printing from consignment pages and accessing full product data, Label Light needs a Lightspeed API bearer token.

Why Do I Need an API Token?

Some Lightspeed pages don't display all product information on screen. The API token allows Label Light to fetch complete product details (including variants, suppliers, and custom fields) directly from Lightspeed's servers.

‚ÑπÔ∏è

What Data is Accessed?

Label Light only reads product and consignment data. It never modifies your Lightspeed data or accesses sensitive information like payment details.

Getting Your API Token

1

Open Lightspeed Developer Tools

In Chrome, while on any Lightspeed Retail page, press F12 or right-click and select "Inspect" to open Developer Tools.

2

Go to the Network Tab

Click the "Network" tab in Developer Tools. If the list is empty, refresh the page.

Currency & Formatting

Configure how prices and currency values appear on your labels.

Currency Symbol

Enter your local currency symbol (e.g., $, ‚Ç¨, ¬£, ¬•). This symbol will be displayed on all price fields on your labels.

Currency Position

Choose where the currency symbol appears relative to the price:

  • Before price - $10.00 (default for USD)
  • After price - 10.00‚Ç¨ (common in European countries)
  • No currency symbol - 10.00 (just the number)

Decimal Rounding

Set how many decimal places to display for price values:

  • No rounding - Shows exact values from API
  • 2 decimal places - Standard for most currencies (e.g., $10.99)
  • 3-4 decimal places - For currencies that use smaller denominations

Label Templates

Label Light supports custom DYMO label templates (.dymo or .label files) created with DYMO Label Software or DYMO Connect.

Uploading Label Templates

  1. In the Label Templates section, click the upload area or drag and drop your label file
  2. Supported formats: .label and .dymo
  3. The template will appear in your template list

Setting a Default Template

Click the star icon (‚≠ê) next to a template to set it as your default. The default template is used when printing labels unless you select a different one.

Designing Label Templates

Create your label templates using DYMO Label Software or DYMO Connect:

  1. Open DYMO Label Software or DYMO Connect
  2. Select your label size (e.g., 30252 Address, 30336 Small, etc.)
  3. Add text objects for each field you want on your label
  4. Name each text object clearly (e.g., "ProductName", "Price", "SKU", "Barcode")
  5. Save the label file
üí°

Naming Objects

Give your label objects meaningful names like "ProductName", "Price", or "SKU". These names are used for field mapping and make it easier to match product data to your label.

Field Mappings

Field mappings define which product data fields from Lightspeed appear in which text objects on your label.

How Field Mappings Work

When you upload a label template, Label Light detects all the text objects (fields) in your label. You can then map each label field to a Lightspeed product data field.

For example:

  • Label field "ProductName" ‚Üí Lightspeed field "name"
  • Label field "Price" ‚Üí Lightspeed field "price_including_tax"
  • Label field "SKU" ‚Üí Lightspeed field "sku"
  • Label field "Barcode" ‚Üí Lightspeed field "supply_price"

Configuring Mappings

  1. Make sure you have a default label template set
  2. Go to the Field Mappings section
  3. For each label field, select the corresponding Lightspeed product field from the dropdown
  4. Click Save Mappings
