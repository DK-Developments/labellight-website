Installation Guide

This guide covers everything you need to install and set up LabelLight for printing labels from Lightspeed Retail.

Prerequisites

Before installing LabelLight, ensure you have the following:

  • Google Chrome - The extension only works with Chrome browser
  • DYMO LabelWriter Printer - Connected to your computer via USB
  • Lightspeed Retail Account - Active subscription with web access
  • LabelLight Subscription - Sign up at our pricing page

Step 1: Install DYMO Connect Software

LabelLight requires DYMO Connect (or DYMO Label Web Service) to communicate with your printer. This software must be running for the extension to work.

  1. Download DYMO Connect

    Visit the DYMO Support page and download DYMO Connect for your operating system.

  2. Install the Software

    Run the installer and follow the on-screen instructions. Accept the default installation options.

  3. Connect Your Printer

    Connect your DYMO LabelWriter printer to your computer via USB. Windows should automatically detect and install drivers.

  4. Verify DYMO Connect is Running

    Look for the DYMO icon in your system tray (Windows) or menu bar (Mac). DYMO Connect must be running whenever you want to print labels.

Step 2: Install LabelLight Extension

From Chrome Web Store (Recommended)

  1. Open Chrome Web Store

    Visit the LabelLight page on the Chrome Web Store (link coming soon).

  2. Click "Add to Chrome"

    Click the blue "Add to Chrome" button to begin installation.

  3. Confirm Installation

    A popup will appear asking for permissions. Click "Add extension" to confirm.

  4. Extension Installed

    The LabelLight icon will appear in your Chrome toolbar. Click it to open settings.

üì∏ Screenshot: Adding LabelLight from Chrome Web Store

Manual Installation (Development)

For developers or testing purposes, you can install the extension manually:

  1. Download the Extension

    Download the extension files from the provided source.

  2. Open Chrome Extensions

    Navigate to chrome://extensions/ in your browser.

  3. Enable Developer Mode

    Toggle on "Developer mode" in the top right corner of the page.

  4. Load Unpacked Extension

    Click "Load unpacked" and select the folder containing the extension files.

Step 3: Sign In to LabelLight

  1. Click the Extension Icon

    Click the LabelLight icon in your Chrome toolbar to open the settings page.

  2. Sign In with Google

    Click the "Sign in with Google" button. A popup will appear for Google authentication.

  3. Authorize the Extension

    Select your Google account and authorize LabelLight to access your basic profile information.

  4. Subscription Verification

    After signing in, the extension will verify your subscription status. Ensure you have an active subscription.

Verifying Installation

To verify that LabelLight is installed and working correctly:

  1. Check Extension Icon

    The LabelLight icon should be visible in your Chrome toolbar. If not, click the puzzle piece icon to pin it.

  2. Open Extension Settings

    Click the icon and verify that the settings page loads without errors.

  3. Check Printer Detection

    In the settings page, go to "General Settings" and verify that your DYMO printer appears in the dropdown.

  4. Test on Lightspeed

    Navigate to Lightspeed Retail. Open a product or consignment page and look for the enhanced "Print Labels" button.

‚úÖ Success

If you can see your printer in the dropdown and the print buttons appear on Lightspeed pages, installation is complete!

Troubleshooting Installation

DYMO Connect Not Running

If printers don't appear, ensure DYMO Connect is running. Look for the DYMO icon in your system tray. If it's not there, launch DYMO Connect from your Start menu.

Extension Not Loading

Try refreshing the Lightspeed page after installing the extension. If problems persist, disable and re-enable the extension in Chrome's extension settings.

Printer Not Detected

If your printer isn't showing up:

  • Check USB connection to the printer
  • Restart DYMO Connect software
  • Try the "Bypass Printer Detection" option in settings (some DYMO Connect versions have a detection bug)

For more troubleshooting help, see the Error Messages guide.
